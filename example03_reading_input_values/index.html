CoderDojo (reading input values)

content

This example demonstrates how to read values from input elements.

Ideas for extensions:

  • Create a calculator consisting of buttons for "0"-"9", "+", "-" and "=". Clicking on a button that is not "=", should append its value to a text field.
  • Create a function that evaluates the expression when you click on the "=" button. Check the eval() function. This function opens a door for hackers if you do not check the text that gets processed by eval(). Why?
