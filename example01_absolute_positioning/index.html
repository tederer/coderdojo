CoderDojo (absolute positioning)

content

This example demonstrates how an element, in our case the blue box, can be positioned anywhere by using Javascript in combination with CSS (Cascading Style Sheets). Use the buttons in the control area to move the blue box and have a look at the code how this works.

Ideas for extensions:

  • Replace the blue box by an image and move it.
  • Create 4 buttons (up, down, left and right) to move the blue box (or image) up, down, left or right by a predefined distance (e.g. 10 pixels).
  • Extend the code to move the blue box along a predefined path. To do this the "setTimeout" function can be used. Searching the internet for "javascript setTimeout" will tell you how this can be used.
